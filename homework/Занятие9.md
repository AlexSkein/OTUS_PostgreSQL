1. Реализовать прямое соединение двух или более таблиц

**Создадим 2 таблицы**

```SQL
CREATE TABLE public.persons (
	id serial4 NOT NULL,
	first_name text NULL,
	second_name text NULL
);

CREATE TABLE public.tickets (
	id serial4 NOT NULL,
	id_person serial4 NOT NULL,
	ticket_number text NULL
);
```
